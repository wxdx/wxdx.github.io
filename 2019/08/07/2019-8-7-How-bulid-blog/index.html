如何用Jekyll搭建属于自己的个人博客网站?

“Let’s Go!”

前言

    虽然市面上有很多blog网站，例如CSDN 、简书等等，但是作为程序猿的我还是想自己搭建一个属于自己的个人博客网站，自己维护。我选用Jekyll搭建自己的博客, Jekyll 的自定制非常容易，基本就是个模版引擎。

环境搭建

Linux环境：

通过RVM安装Ruby环境

1、安装gpg密钥

– 如果出现：
gpg: keyserver receive failed: Syntax error in URI
请再次执行一遍上面的指令

2、安装RVM

  • 如果最后一行出现如下提示（让你捐赠）：
    👉  Donate: [https://opencollective.com/rvm/donate](https://opencollective.com/rvm/donate)
说明RVM安装完成了

3、更新bash的环境配置

  • 若出现以下提示，则说明RVM安装成功

5、安装Ruby2.5.5

  • 若依次出现以下提示，则说明Ruby安装成功

6、更改gem源（国外源太慢了，卡到你怀疑人生）

7、安装Jekyll

  • 若出现以下提示，则说明Jekyll安装成功

在Github上寻找Jekyll个人博客模板

     Github上有很多优秀的Jekyll的博客模板，我挑选了一个Star最多的项目—> Jekyll-now

1、下载git

2、检出Jekyll-now

3、安装Jekyll-now所需的gem依赖

4、运行Jekyll

  • 如果服务器是云服务器并且拥有外网IP

  • 如果服务器是本地机器

    5、访问浏览器

http://127.0.0.1:4000

搭建成功！！

个性化设置

     请参照jekyll-now的README，主要是修改_config.yml的配置，非常简单！
